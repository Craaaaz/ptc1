# 玩狗大特

這是一個使用 Godot Engine 製作的遊戲專案，包含多個 GDScript 腳本、場景與美術資源。

## 專案結構

- `GDscript/`：主要遊戲腳本（如主攝影機、地圖設置、玩家移動等）。
- `Images/`：遊戲內使用的圖片資源（角色、地形、物件等）。
- `Scenes/`：Godot 場景檔案（未列出細節）。
- `project.godot`：Godot 專案設定檔。
- 其他：說明文件、圖示等。

## 開發環境

- Godot Engine 4.x
- 建議使用 VS Code 或 Godot 內建編輯器進行腳本開發

## 執行方式

1. 下載本專案並以 Godot Engine 開啟 `project.godot`。
2. 執行主場景（請依專案設定選擇主場景）。
